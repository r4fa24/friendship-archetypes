# Friendship Style Discovery Tool 🤝✨
An interactive personality assessment tool that identifies unique friendship "vibes" based on user behavior and responses.

## 🚀 Technical Highlights
* **Dynamic Form Logic:** Utilizes JavaScript to process user inputs and deliver real-time results based on specific personality archetypes.
* **Semantic HTML5:** Built with a clean, accessible structure including custom form elements and informative sections.
* **Responsive UI:** Designed with a mobile-first approach using flexible viewports to ensure high performance on all devices.
* **Archetype Mapping:** Implemented logic for four distinct friendship profiles: The Anchor, The Spark, The Joker, and The Lone Wolf.

## 🛠️ Tech Stack
* **Front-end:** HTML5, CSS3
* **Logic:** JavaScript (ES6+)
